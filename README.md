仓库仓位查询系统
这是一个基于HTML、Tailwind CSS和JavaScript开发的仓库仓位查询系统，支持快速登录、多用户管理、仓位查询和录入等功能。

系统功能
1. 用户认证
快速登录: 点击按钮即可快速登录
多用户支持: 管理员和普通用户角色
2. 仓位管理
仓位查询: 支持编码、名称、仓位多条件组合查询
二维码扫描: 支持单个和批量二维码扫描查询
仓位录入: 支持二维码录入和手动录入
数据可视化: 直观的结果展示和详情查看
3. 系统管理
用户管理: 管理员可管理用户账号（仅管理员）
系统设置: 配置二维码格式等系统参数（仅管理员）

页面说明
主要页面
index.html - 快速登录页面
search.html - 查询页面
result.html - 查询结果页面
input.html - 仓位录入页面
users.html - 用户管理页面（仅管理员）
settings.html - 系统设置页面（仅管理员）
使用说明
使用方法
访问 index.html 页面
点击对应的用户角色按钮快速登录
进入查询页面进行操作
技术特点
纯前端实现: 使用HTML、CSS、JavaScript开发
响应式设计: 适配各种屏幕尺寸
本地存储: 使用localStorage和sessionStorage存储数据
二维码支持: 集成jsQR库实现二维码扫描
现代化UI: 使用Tailwind CSS构建美观的用户界面
浏览器要求
支持现代浏览器（Chrome、Firefox、Edge等）
启用JavaScript
允许本地存储（localStorage/sessionStorage）
如需使用摄像头功能，请允许摄像头访问权限
故障排除
常见问题
无法登录: 检查浏览器存储权限，使用访客登录页面
摄像头不工作: 检查摄像头权限设置
数据丢失: 使用测试工具重新创建数据
调试方法
打开浏览器开发者工具（F12）
查看控制台（Console）标签页的错误信息
检查应用（Application）标签页的存储数据
使用测试工具进行诊断
注意事项
本系统使用浏览器本地存储，清除浏览器数据会导致系统数据丢失
摄像头功能需要HTTPS环境或本地访问
管理员账号拥有所有权限，请妥善保管
建议定期备份重要数据
